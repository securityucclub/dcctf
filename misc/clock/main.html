Clock
Welcome to the clock.

Every second you stay on this website the current time is hashed with MD5 and compared to a stored hash that we selected previously.

There is only one second in time that represents the correct time. If you are looking at the webiste at the correct moment, you will get a message reading "this is the correct time".

The flag is DCCTF{th3_c0rr3ct_t1m3_is_XX:YY:ZZ} where you have to replace XX with the hours, YY with the minutes and ZZ with the seconds of the correct time.
